1. アクセスとログイン

- ブラウザで http://150.89.233.201 にアクセスします。
- ログインフォームが表示されるので、いずれかのアカウントでログインします。
- ログインに成功するとゲーム選択画面やマッチング画面にリダイレクトされます。

2. ゲームの選択と参加

- トップページまたはメニューからプレイ可能なゲームのリンクをクリックします。
- 「マッチング」ボタンが表示されるので、それをクリックして対戦者を探す。
- マッチング機能を利用する場合は、相手が見つかるまで待機し、相手が見つかると対戦画面に遷移します（`matching.html`, `match_success.html` などを参照）。

3. 対局中の操作（一般的な流れ）

- 自分の手番が来たら画面上の操作（クリックやボタン）で手を選択します。
- 選択後、相手の手が決まるまで待ちます。結果は即時更新されます。
- 対局終了時には勝敗が表示され、再戦やメニューへ戻るオプションが表示されます（`match_result.html` 参照）。

4. 観戦機能の利用

- 本マニュアルの2項目目と同様、ゲームを選択すると観戦ボタンが表示されます。
- ボタンをクリックすると現在行われている対戦が一覧で表示されます。
- 任意の対戦を選び、観戦をクリックすると観戦用ページ（例: `marubatsu_spectate.html`, `gomoku_spectate.html`）に移動して現在の対局を観戦できます。
- 観戦中は操作が制限され、観戦用インターフェースで盤面やターン、ログ等を確認できます。

5. よくある操作
- ログイン: トップページのログインリンク
- ゲーム選択: トップページまたはメニューからゲームページへ
- マッチング: マッチング画面で「対戦相手を探す」ボタン
- 観戦: 各ゲームの "_spectate.html" ページ
- ログアウト: 画面上のログアウトボタンをクリック
